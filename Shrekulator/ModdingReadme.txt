To add conversion categories, simply create a new textfile with a '.udef' file-extension, which
will contain data in a JSON format.

On the 'FormatString' property: http://www.independent-software.com/net-string-formatting-in-csharp-cheat-sheet.html might come in handy.
The index 0 is the calculated amount, index 1 is the display name and index 2 is the unit's symbol
